A modelagem √© feita sobre a s√©rie estacion√°ria, e n√£o sobre a s√©rie n√£o estacion√°ria original.
2.  **Previs√£o:** Permite obter previs√µes para a s√©rie original a partir da previs√£o das mudan√ßas na s√©rie, utilizando um processo recursivo. A previs√£o para a s√©rie original, com raiz unit√°ria, apresenta maior incerteza para horizontes mais longos, e segue uma trajet√≥ria com deriva.
3. **An√°lise de Res√≠duos**: A an√°lise de res√≠duos √© realizada sobre os res√≠duos do modelo da s√©rie transformada (diferen√ßas), que √© uma s√©rie estacion√°ria. Os testes de autocorrela√ß√£o e de raiz unit√°ria s√£o aplicados aos res√≠duos do modelo da s√©rie diferenciada.
4.  **Interpreta√ß√£o Econ√¥mica:** Permite interpretar a din√¢mica das mudan√ßas na s√©rie em termos de choques e respostas a esses choques. Os coeficientes do modelo para a s√©rie diferenciada podem ter interpreta√ß√µes econ√¥micas importantes. A an√°lise do componente estacion√°rio permite capturar os efeitos de curto prazo que podem ter efeitos persistentes no longo prazo. A estima√ß√£o da deriva do processo √© tamb√©m um ponto de interesse.
5.  **Testes de Raiz Unit√°ria:** A exist√™ncia de uma raiz unit√°ria na s√©rie original implica que a s√©rie diferenciada seja estacion√°ria, o que pode ser verificado por testes de raiz unit√°ria sobre a s√©rie diferenciada. A n√£o rejei√ß√£o da hip√≥tese nula de raiz unit√°ria na s√©rie original e a rejei√ß√£o da hip√≥tese nula sobre a s√©rie diferenciada, confirmam a exist√™ncia da raiz unit√°ria na s√©rie original.
6.  **Compara√ß√£o de Modelos:** Os modelos para a representa√ß√£o estacion√°ria podem ser comparados atrav√©s de crit√©rios de informa√ß√£o, como AIC e BIC. A an√°lise da adequa√ß√£o do modelo, como a an√°lise dos res√≠duos, permite verificar a validade do modelo.
> üí° **Exemplo Num√©rico:**
>
> *  **Modelagem:** Se o teste de raiz unit√°ria indicar que uma s√©rie de pre√ßos de a√ß√µes possui raiz unit√°ria, podemos modelar a mudan√ßa no pre√ßo da a√ß√£o (retornos) como um processo AR(1), utilizando m√©todos de estima√ß√£o de modelos estacion√°rios. O modelo seria $\Delta \log(Preco_t) = \phi \Delta \log(Preco_{t-1}) + \epsilon_t$.
>
>   ```python
>   import numpy as np
>   from statsmodels.tsa.arima.model import ARIMA
>
>   # Suponha que temos os seguintes retornos (primeira diferen√ßa do log dos pre√ßos)
>   retornos = np.array([0.02, -0.01, 0.03, 0.01, -0.02, 0.04, 0.01, -0.03])
>
>   # Ajustar um modelo AR(1)
>   model = ARIMA(retornos, order=(1, 0, 0))
>   model_fit = model.fit()
>   print(model_fit.summary())
>    # O output mostrar√° os coeficientes do AR(1)
>   ```
> *   **Previs√£o:** As previs√µes de longo prazo da s√©rie original seguem uma trajet√≥ria com uma deriva. As previs√µes da s√©rie original podem ser obtidas por meio da integra√ß√£o das previs√µes das mudan√ßas na s√©rie, obtidas usando o modelo ARMA. Se a previs√£o para a mudan√ßa na s√©rie for dada por $\hat{\Delta y}_{t+1} = \mu + \phi \hat{\Delta y}_t$, ent√£o a previs√£o para a s√©rie original √© dada por $\hat{y}_{t+1} = y_t + \hat{\Delta y}_{t+1}$.
>    ```python
>   # Suponha que o modelo estimado √©:  Œîy_t = 0.01 + 0.5 Œîy_{t-1} + Œµ_t
>   # e o √∫ltimo valor observado da s√©rie original √© y_t = 100
>   y_t = 100
>   delta_y_t = retornos[-1] # √∫ltimo valor da s√©rie de diferen√ßas
>
>   mu = 0.01
>   phi = 0.5
>
>   # Previs√£o para a mudan√ßa em t+1
>   delta_y_t_mais_1_prev = mu + phi * delta_y_t
>
>   # Previs√£o para o valor da s√©rie original em t+1
>   y_t_mais_1_prev = y_t + delta_y_t_mais_1_prev
>
>   print(f"Previs√£o para a mudan√ßa em t+1: {delta_y_t_mais_1_prev:.4f}")
>   print(f"Previs√£o para a s√©rie original em t+1: {y_t_mais_1_prev:.4f}")
>   ```
> * **Interpreta√ß√£o Econ√¥mica:** Se o modelo para a primeira diferen√ßa de uma s√©rie temporal indicar que $\Delta y_t = \delta + 0.4\Delta y_{t-1} + \epsilon_t$, significa que a mudan√ßa na s√©rie √© influenciada por mudan√ßas passadas, e o coeficiente 0.4 representa a persist√™ncia do efeito da mudan√ßa passada sobre a mudan√ßa atual. O termo $\delta$ representa a deriva, que √© o crescimento m√©dio da s√©rie.
> *   **Testes de Raiz Unit√°ria:** Se o teste de raiz unit√°ria indicar que uma s√©rie original possui raiz unit√°ria, o teste aplicado √† primeira diferen√ßa deve rejeitar a hip√≥tese nula de presen√ßa de raiz unit√°ria.
>
>  ```python
>  from statsmodels.tsa.stattools import adfuller
>  # Suponha que a s√©rie original seja 'y' e a diferenciada seja 'delta_y'
>  y = np.array([10, 12, 15, 18, 20, 23, 25, 28])
>  delta_y = np.diff(y) # calcula a primeira diferen√ßa
>
>  # Teste ADF na s√©rie original
>  adf_result_y = adfuller(y)

>  adf_result_delta_y = adfuller(delta_y)
>  print(f"ADF Estat√≠stica para delta_y: {adf_result_delta_y[0]:.4f}, p-valor: {adf_result_delta_y[1]:.4f}")
>   # Se a s√©rie original tiver raiz unit√°ria, o p-valor ser√° > 0.05 e a s√©rie diferenciada ter√° p-valor < 0.05
>  ```
> *   **Compara√ß√£o de Modelos:** Ao modelar a diferen√ßa de uma s√©rie temporal, podemos comparar diferentes modelos ARMA, utilizando crit√©rios como AIC e BIC, e escolhendo o modelo com menor valor de crit√©rio.
>  ```python
>  from statsmodels.tsa.arima.model import ARIMA
>  import pandas as pd
>  # Suponha que a s√©rie diferenciada seja delta_y
>  delta_y = np.array([2, 3, 3, 2, 3, 2, 3])
>  # Criar um dataframe pandas para facilitar os c√°lculos do AIC e BIC
>  data = pd.Series(delta_y)
>  # Modelos para comparar
>  models = [ARIMA(data, order=(1, 0, 0)),  # AR(1)
>            ARIMA(data, order=(0, 0, 1)), # MA(1)

>  results = []
>
>  for model in models:
>      model_fit = model.fit()

>  print("| Modelo  | AIC    | BIC    | Parameters |")
>  print("|---------|--------|--------|------------|")
>  for aic, bic, order in results:
>       print(f"| ARMA{order} | {aic:.2f} | {bic:.2f} |{order} |")
>  # O modelo com menor AIC e BIC ser√° o preferido
>  ```
